# Pay UI

`alipay-payui` 是一套适用于支付场景的组件库。

## 特性

- `低成本` 接入方便，无需额外配置
- `兼容性` 自动适配支付宝、微信等app设计及业务体系，完美融入业务

## 版本

[![npm](https://img.shields.io/npm/dt/alipay-payui.svg?style=flat-square)](https://www.npmjs.com/package/alipay-payui)

- IOS > 8
- Android > 4
